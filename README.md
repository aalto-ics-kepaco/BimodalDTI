# BimodalDTI
Integrating Language Models and 3D Protein-Ligand Complex Structures for Drug-Target Binding Affinity Prediction

# Summary of BimodalDTI model
BimodalDTI fuses pre-trained embeddings from biochemical sequence-based langauge models with 2 geometric graph neural networks (GNNs) operating on Boltz-1 predicted structures for drug-target complexes, namely an atom-level 3D GNN and a protein language model-enhanced ligand atom-protein residue level 3D GNN, for accurate DTI prediction.
